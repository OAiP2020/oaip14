Идея алгоритма
Дано: Массив целых чисел из n элементов  и целое число a.
Требуется: Найти номер элемента массива, раавого числу a. Если таких жлементов несколько- найте первый из них. Если такого элемента нет в массиве - вывести соответствующее сообшение.
Идея алгоритма.
Нужный элемент ишется путём последовательного перебора всех элементов массива, начиная с первого. Каждый элемент сравнивается с заданным числом a. Если элемент найден выводится номер соответствующего элемента массива. Если элемент не найден - выводится соответствующее сообшение.
Сложность алгоритма - O(n)
